# VigiSalud - MVP

App de predicción de consultas para salud pública. Proyecto Humai.

## Tecnologías
- Python 3 + numpy + requests
- Supabase (PostgreSQL)
- Termux (Moto G65)
- Azure App Service

## Scripts
- ingesta.py - Datos simulados
- datos_reales.py - World Bank API
- datos_extra.py - Datos estacionales
- promedio.py - Modelo predictivo

## Dashboard
Supabase Table Editor
